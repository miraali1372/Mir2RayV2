## چرا Mir2rayV2؟

- اتصال سریع و ساده برای استفاده روزمره
- تست و مرتب‌سازی کانفیگ‌ها برای پیدا کردن گزینه‌های بهتر
- نمایش IP و DNS فعلی روی صفحه اصلی
- ابزارهای DNS و نمایش ترافیک زنده
- پشتیبانی از چند منبع کانفیگ
- مناسب برای شبکه‌های شلوغ و ناپایدار

## آموزش استفاده

اگر تازه برنامه را نصب کرده‌اید، این مسیر را بروید:

### 1) برنامه را نصب و باز کنید

- روی دکمه دانلود بالا بزنید.
- فایل APK را نصب کنید.
- اگر گوشی اجازه نصب از منابع ناشناس خواست، آن را فعال کنید.
- برنامه را باز کنید.

### 2) کانفیگ بگیرید

- از بخش دریافت کانفیگ، کانفیگ‌ها را از منبع دلخواه بگیرید.
- اگر خواستید، منبع کانفیگ را عوض کنید تا لیست‌های مختلف را امتحان کنید.
- بعد از دریافت، صبر کنید تا لیست کامل شود.

### 3) کانفیگ‌ها را تست کنید

- وارد بخش کانفیگ‌ها شوید.
- روی دکمه `تست دانلود` بزنید.
- برنامه همه کانفیگ‌ها را بررسی می‌کند و در پایان، بهترین‌ها را بالاتر می‌آورد.
- کانفیگ‌هایی که عملاً سرعت ندارند، در پایان کنار گذاشته می‌شوند.

### 4) بهترین کانفیگ را انتخاب کنید

- کانفیگی را انتخاب کنید که سرعت و پاسخ بهتری داشته باشد.
- اگر یک کانفیگ وصل نشد، سراغ مورد بعدی بروید.
- معمولاً کانفیگ‌های بالای لیست، گزینه‌های بهتری هستند.

### 5) وصل شوید

- روی دکمه `Connect` بزنید.
- برنامه تلاش می‌کند همان کانفیگی را که انتخاب کرده‌اید وصل کند.
- اگر شبکه شلوغ باشد، چند ثانیه صبر کنید تا اتصال کامل شود.

### 6) وضعیت اینترنت را چک کنید

- صفحه اصلی، IP و DNS فعلی را نشان می‌دهد.
- اگر خواستید ببینید اینترنت شما دقیقاً از چه مسیری می‌رود، همان صفحه کمک می‌کند.

### 7) اگر اینترنت کند بود

- یک کانفیگ دیگر را امتحان کنید.
- دوباره تست دانلود بگیرید.
- اگر DNS مناسب‌تری دارید، همان را استفاده کنید.
- در شبکه‌های شلوغ، همیشه یک کانفیگ بهتر از بقیه جواب می‌دهد.

## هر بخش چه کاری می‌کند؟

- **صفحه اصلی**: خلاصه وضعیت، IP فعلی، DNS فعلی و دسترسی سریع
- **کانفیگ‌ها**: دریافت، تست، مرتب‌سازی و انتخاب کانفیگ
- **DNS**: بررسی DNS و تست دانلود روی مسیرهای مختلف
- **اتصال**: وصل شدن به کانفیگ انتخاب‌شده

## نکته‌های مهم برای استفاده بهتر

- همیشه قبل از اتصال، یک یا دو کانفیگ را تست کنید.
- اگر سرعت پایین بود، فقط به اولین نتیجه اکتفا نکنید.
- در اینترنت ناپایدار، چند ثانیه بیشتر صبر کردن بهتر از قطع و وصل سریع است.
- برای نتیجه بهتر، از نسخه آخر برنامه استفاده کنید.
