## ✨ امکانات

- ⚡ **هسته‌ی Xray** — پشتیبانی از VLESS، VMess، Trojan و Shadowsocks با ترنسپورت‌های TCP/WS/gRPC/HTTP2/Reality
- 🇮🇷 **روتینگ هوشمند ایران** — سایت‌ها و CDNهای داخلی (ArvanCloud، Derak، IranServer، ParsPack) مستقیم رد می‌شوند؛ دیتابیس روتینگ قابل‌به‌روزرسانی از داخل برنامه
- 🚀 **انتخاب خودکار بهترین کانفیگ** — تست پینگ و دانلود، رتبه‌بندی، و سوییچ خودکار (failover) در صورت قطعی
- 🔇 **مسدودسازی QUIC** برای روان‌تر شدن یوتیوب و مرور
- 🧩 **FakeDNS و DNS رمزنگاری‌شده (DoH)** — اختیاری، برای باز شدن سریع‌تر و جلوگیری از دستکاری DNS
- 🛡️ **محافظت در برابر نشت (Kill Switch)** از طریق Always-on VPN سیستم
- 📡 **ابزار DNS و اسکنر CDN** برای یافتن سریع‌ترین مسیر
- 🔄 **به‌روزرسانی درون‌برنامه‌ای** — آخرین نسخه را مستقیم از همین صفحه می‌گیرد

## 📥 نصب

1. از بخش [Releases](https://github.com/miraali1372/Mir2RayV2/releases/latest) فایل `Mir2rayV2-*.apk` را دانلود کنید.
2. هنگام نصب، اجازه‌ی «نصب از منابع ناشناس» را بدهید.
3. برنامه روی **اندروید ۷.۰ به بالا** و همه‌ی معماری‌ها (arm64, arm-v7a, x86, x86_64) نصب می‌شود.
