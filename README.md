## چرا Mir2rayV2؟

- اتصال سریع و ساده برای استفاده روزمره
- تست و مرتب‌سازی کانفیگ‌ها برای پیدا کردن گزینه‌های بهتر
- نمایش IP و DNS فعلی روی صفحه اصلی
- ابزارهای DNS و نمایش ترافیک زنده
- پشتیبانی از چند منبع کانفیگ
- مناسب برای شبکه‌های شلوغ و ناپایدار

## این برنامه برای چه کسی ساخته شده؟

برای کسی که می‌خواهد بدون شلوغی اضافه، سریع‌تر وصل شود، کانفیگ مناسب‌تری پیدا کند و قبل از استفاده، کیفیت مسیر را بهتر بسنجد.

## نصب

1. روی دکمه دانلود بالا بزنید.
2. فایل APK را نصب کنید.
3. اگر نصب از منابع ناشناس خواست، اجازه بدهید.
4. برنامه را باز کنید و کانفیگ دلخواه را انتخاب کنید.
